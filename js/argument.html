argument 객체 -유사배열

인자 - argument function a(arg){ } a(1) 1은 인자/아규먼츠 , arg는 매개변수/파라미터 임. arguments는 자바스크립트에 내장되어있는 변수명인데, 이 안에는 유사배열이 담겨져있다. 입력한 인자를 배열로 .. a+=1은 a=a+1 arguments라는 내장 변수의 예제에서의 역할 - arguments라는 객체를 포함하고 있는 함수가 몇개의 인자를 가지고있는지 알 수 있음 (arguments.length) -arguments[i]를 통해 인덱스값을 입력함을 통해서 인자의 특정 자리수의 값이 무엇인지 알아낼 수 있음.
